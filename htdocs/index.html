Real Time Web Logging in Linz (リアルタイムウェブロギン・イン・リンツ)

Node感に注目して、新しい体験の可能性を探る実験サイト「Real Time Web Logging (リアルタイムウェブロギン)」。
オーストリア・リンツ市で開かれるメディアアートの祭典「アルスエレクトロニカ・フェスティバル」の開催期間に合わせて、ユニバ株式会社が公開しています。

リアルタイムウェブロギンのサーバサイドプログラムは、iPhone/iPadのセンサーが感知する加速度、地磁気、GPS電波強度などのデータを、Node.jsとSocket.IOによるリアルタイム通信の技術を使って、秒間数十回の頻度で配信します。
リアルタイム・データは、APIを通じて一般公開されるので、誰でも利用することができます。

リアルタイム・データを解釈するヴィジュアライジング・プログラムは、ユニバ株式会社のリンツ滞在スタッフ、東京滞在スタッフがそれぞれアルス・エレクトロニカ・フェスティバルの期間中(9月1日～9月6日予定)に開発し、公開していきます。

これまで誰も知らなかった体験ができますように！

Real Time Web Logging @ Tokyo (リアルタイムウェブロギン・アット・東京)

9月2日～4日の期間中、東京都渋谷区でリアルタイムウェブロギン・イン・リンツと連動したパーティーを実施します。
リアルタイム・データ・ヴィジュアライジングを複数展示するほか、公開開発、リンツ滞在スタッフとのビデオチャットなどを行います。遊びに来てくださいね！
